# 列表

## 无序列表

用* 号

* 111

* 222

* 333
  用+号
+ 111

+ 222

+ 333
  用- 号
- 111

- 222

- 333

## 有序列表

有序列表使用数字并加上 . 号来表示[^1]

1. 1111

2. 2222

3. 3333
   
   ## 列表嵌套
   
   列表嵌套只需在子列表中的选项前面添加两个或四个空格即可

4. 第一项：
   
   - 第一项嵌套的第一个元素
   - 第一项嵌套的第二个元素

5. 第二项：
   
   - 第二项嵌套的第一个元素
   - 第二项嵌套的第二个元素

[^1]: 数字可不按顺序
