# 输入与输出

## 输出

```python
print(~~~)
```

+ 多个数据可用`,`或`+`拼接(`,`比`+`多个空格)
  +不同type数据不可直接拼接
  
  ### 字符串格式化
  
  __1.`"aaa%sAAA"%(111)`__
  
  > %s 把()转为str放入%s中
  > %d 把()转为int放入%d中
  > 
  > > m,n控制float的宽度与精度
  > 
  > %f 把()转为float放入%f中
  
  __2.`f"啊啊啊啊{变量}啊啊啊啊"`___
